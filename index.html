協調運動障害下での
スクーバダイビング実施の可能性を探る

目的

協調運動障害下でのスクーバダイビングの可能性を探る

身体的前提条件

  1. 現在協調運動障害で 3 級の手帳を持っている
  2. 歩行時に急激な方向転換をする際に、バランスを崩しそうになり、ふらつく
  3. バランスを崩しそうなシーンは、方向転換時に限られる。
  4. 現在転倒予防のために登山用ストックを利用している。
    転倒を予防できるように、ちょこちょこと地面にストックを当てながら、バランスを取って歩行している
    (あくまで転倒予防でがっつりストックに体重を預けたりはしていない)
  5. 上記以外の所見は特になし。

予想されるリスクシーン

現在の障害状態の特質から、リスクは水中ではなく、ボート上で高くなると予測される。

  1. 港でのボート乗船、下船のタイミングでの落水
    埠頭に腰掛けて上下船するなど工夫すれば予防可能だと思われる。
  2. 港でのボートに乗船、下船のタイミングで、埠頭とボートの間に転落して体ごと挟まれる
    通常埠頭とボートの間に大きな隙間はないので、この問題は起こらないのではないか。
  3. ポイント、港間の移動中のボートからの落水
    ボートの床に座っていれば発生しない。
  4. ボート上での機材の装着中、脱着中における、ボートからの落水
    もしもの場合に備えて被害を低減する方法が考えられる。
    エントリーとエグジット方法の検討を参照。
  5. エグジット時のラダーからの落水
    もしもの場合に備えて被害を低減する方法が考えられる。
    エントリーとエグジット方法の検討を参照。

エントリーとエグジットの方法の検討

前提条件

  1. ダイビングポイントに、あまり流れがないこと
    流れがある環境の場合は今後の研究課題。
  2. 穏やかな海況であること
    それ以外の環境の場合は今後の研究課題。

エントリー方法の検討

  1. スクーバユニットはボート上で組み立ててバルブを開け、BC には吸気しておく。
  2. スクーバユニットは上記状態でエントリー前に予め水面に投入する。
  3. その際、ロープ等でボートと繋げておくことが望ましい。
    (ベタ凪、流れなしだと省略可能かも)
  4. ウェイトベルトと 3 点セットを身に着けた状態で、エントリーに適した方法でエントリーする。
    バックロール、フロントロール、サイドロール、静かに足から入水など状況により適切に選択する。
  5. 水面でスクーバユニットを装着
  6. 潜降

エグジット方法の検討

  1. 浮上
  2. ボートからスクーバユニット用にロープを流しておいてもらう。
  3. スクーバユニット、ウェイトベルトを水面で脱装し、ロープに固定する。
  4. 軽機材のみでラダーを登ってエグジット。
    フィンは脱いでエグジットした方がいいかも。

ダイビング後

通常のログブック以外に、以下を後日整理してまとめる。

  1. ダイビングサービスでの課題と注意点
  2. ボートへの移乗の課題と注意点
  3. エントリー時の課題と注意点
  4. 潜降時の課題と注意点
  5. 潜水中の課題と注意点
  6. 浮上時の課題と注意点
  7. 安全停止時の課題と注意点
  8. エグジット時の課題と注意点
  9. ボートから陸への移乗と注意点
  10. 全体のまとめ
